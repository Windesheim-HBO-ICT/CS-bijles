Opdracht Printer

voor deze opdracht kan je het project calculator als voorbeeld gebruiken.

Printer a: definieer een delegate Print die een string krijgt en een void teruggeeft.

Printer b: maak een methode PrintUit die een Print delegate en een string als argumenten heeft en een void als uitput. De methode moet de delegate aanroepen met de string als argument.

Printer c: definieer 2 of meer methodes die aan de delegate voldoen. Ze krijgen een string en printen het uit op de console.  Zorg ervoor dat deze methodes verschillen van elkaar.

Printer d: roep PrintUit aan met dezelfde string en elke functie die je hebt geschreven en kijk wat de PrintUit op de console print.
